Variables

変数を変更して出力の結果を変更できます。

Variables Type Defalut Description
$libsass-version Number 3.4 libSassのバージョンを指定します。そのままでも問題ありません。
$ruby-sass Boolean false RubyのSassを使っているか指定します。そのままでも問題ありません。
$postcss-assets Boolean false postcss-assetsを使っているか指定します。
$url-replacement String './assets/images/' 画像を保存しているパスを指定します。
$colors-map Map - キーにキーワード、値に色を指定します。入れ子もできます。
$screen Map - ブレイクポイントを指定します。
$z-map-reverse Boolean false $z-mapのMapの順序を逆にします。
$z-map Map - z-indexの順序を指定します。キーにキーワード、値はなんでもいいので文字列を入れてください。
$debug-outlining Boolean false デバッグ用にアウトラインを表示するか。

Media Queries

メディアクエリーを簡単に設定できます。
そのまま利用すると同じ@mediaルールが乱立してしまうので、node-css-mqpackerといったメディアクエリーをまとめるプラグインなどと併用して使用してください。

@include mq($size, $width: max) {...}
Name Type Defalut Description
$size String or Number - メディアクエリーのサイズ。$screen変数のキーか数値をpxで指定します。
$width String max max か minを指定。

関連する変数

$screen: (
  sm: 544px,
  md: 749px,
  lg: 992px
) !default;

Example

.foo {
  @include mq(sm) {
    content: 'max-width: 544px';
  }
  @include mq(md, min) {
    content: 'min-width: 748px';
  }
  @include mq(1200px) {
    content: 'max-width: 1200px';
  }
  @include mq(1200px, min) {
    content: 'max-width: 1199px';
  }
}

Media Queries: Only

メディアクエリーの範囲を指定します。

@mixin mq-only($min-width, $max-width) { ... }
Name Type Defalut Description
$min-width String or Number - 最小幅を指定します。$screen変数のキーか数値をpxで指定します。省略も可能です。
$max-width String or Number - 最大幅を指定します。$max-widthの値を省略してキーを指定すると、一つ前のキーが$min-widthになります。

関連する変数

$screen: (
  sm: 544px,
  md: 749px,
  lg: 992px
) !default;

Example

.foo {
  @include mq-only(sm) {
    content: '';
  }
  @include mq-only(md) {
    content: '';
  }
  @include mq(sm, lg) {
    content: '';
  }
  @include mq(300px, md) {
    content: '';
  }
  @include mq(890px, 1200px) {
    content: '';
  }
}
